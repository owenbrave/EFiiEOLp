# 项目简介

## 核心功能
- 基于YOLOv8的扬尘实例分割系统
- 支持图片、视频及摄像头进行检测
- 支持选择不同模型
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存
- 支持自定义置信度和IOU阈值
- 可选择显示标签和原图

## 适用场景
- 环境监测与扬尘检测
- 实时视频分析
- 批量图像处理
- 科研实验与模型训练

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 基于Python 3.9，依赖opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 包含1192张标注数据集，专注于扬尘(DUST)一类检测
- 提供完整源码、训练好的模型、数据集及运行教程
- 配置成功环境可直接使用，无需额外训练
